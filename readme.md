* 스프링 생명주기
  - 싱글톤 ➡️ 스프링부트앱이 꺼지기 전까지 살아있음.
  - 세션 ➡️ 브라우저당 객체가 1개씩 만들어짐.
  - 리퀘스트 ➡️ 요청당 객체가 1개.
  - 프로토타입 ➡️ 매번 새로 만든다.
